# One Pager 工作流（finance-one-pager）

1. **解析输入**：公司名、证券代码、市场（A/港/美）；缺代码时推断或向用户确认。
2. **调用 Tushare**：`stock_basic` → `daily`（近 1 年）→ `fina_indicator`、`income`（如 20220101–当前），提取营收、增速、利润率、ROE/ROA、杠杆等。
3. **填充模板**：将数据填入 0. 基本信息 / 1. 业务 / 2. 财务摘要 / 3. 亮点 / 4. 风险；无法取数部分用定性描述并注明。
4. **输出**：按 SKILL.md 中「上市公司 One Pager 模板」生成完整一页纸，附数据来源与免责声明。

数据优先从 Tushare 获取；无权限或缺失时降级为定性并标注「基于历史公开信息/非实时财报」。
